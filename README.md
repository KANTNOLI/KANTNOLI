## 🚀 Featured project: KGEngine

**A high‑level 3D engine on top of THREE.js**  
- Write less, create more – simplified API  
- **Native HTML/CSS elements inside WebGL** with real depth, occlusion and camera interaction  
- Built‑in shaders, ready‑to‑use components  
- Framework‑agnostic (React, Vue, Svelte, Angular, vanilla)
